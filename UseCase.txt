# Сценарий просмотра информации на сайте

## __Цель:  Ознакомление с информацией о пиццах на сайте.__

## Акторы: 

> -потенциальный клиент.
> -система.

### Основной сценарий:

- пользователь открывает  страницу сайта в любом из браузеров.
- на странице отображается информация (название, фото, состав, вес, стоимость, возможные заменяемые/ удаляемые ингредиенты) о всех 7-ми пиццах последовательно сверху вниз в алфавитном порядке по названиям. 
- пользователь скроллит экран (вверх / вниз) для ознакомления с информацией о пиццах.
 
### _Альтернативный сценарий(дополнительный):_

- пользователь переходит на станицу сайта по deeplink  с внешнего сайта. 
- на странице отображается информация (название, фото, состав, вес, стоимость, возможные заменяемые/ удаляемые ингредиенты) о всех 7-ми пиццах последовательно сверху вниз в алфавитном порядке по названиям. 
- пользователь скроллит экран (вверх / вниз) для ознакомления с информацией о пиццах.
